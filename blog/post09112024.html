Steganography

Date: September 11th, 2024

I came across Steganography while I was at graduate school learning about Cryptography. Steganography is the science of hiding information by embedding a secret message within an object (Meghanathan, 2015). The object can be in the physical world; for example, writing a message in lemon juice then revealing the message with heat. The object can also be in the electronic space where a text message is written to a PNG formatted image.


Steganography is similar to Cryptography because both fields attempt to hide information from unsolicited viewers. The most significant difference between the two is that Steganography does not attempt to hide the object with the embedded information. For an object such as a PNG image with an embedded message, the image may be viewable online, but it is not obvious that it contains a hidden message.
